Il Muro — Setup Firebase

Per il sync in tempo reale hai bisogno di un progetto Firebase gratuito.
Segui questi passaggi (5 minuti):

  1. Vai su console.firebase.google.com e crea un progetto
  2. Nel progetto, vai su Firestore Database → Crea database → Modalità test
  3. Vai su Impostazioni progetto → Le tue app → icona Web </>
  4. Registra l'app e copia i valori firebaseConfig
  5. Aprì questo file HTML in un editor e incolla i valori nella variabile FIREBASE_CONFIG in cima
